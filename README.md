# JBM
 
- 基于java spring boot 模块化企业级框架
- 主要由util;autoconfig;webjars等模块组成


技术选型
● 核心框架：Spring Framework + Dubbo
● 安全框架：Apache Shiro
● 任务调度：Spring + Quartz
● 持久层框架：MyBatis + MyBatis-Plus
● 数据库连接池：Alibaba Druid
● 缓存框架：Redis
● 会话管理：Spring-Shiro + redis
● 日志管理：SLF4J、Log4j2
● 前端框架：Angular JS + Bootstrap + Jquery>
